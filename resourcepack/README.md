# GemPlugin Resource Pack

This is the official resource pack for the GemPlugin Minecraft plugin. It provides custom textures for all gem items.

## Installation

1. Download or copy the entire `resourcepack` folder
2. Place it in your Minecraft `resourcepacks` folder
3. Enable it in Minecraft's Resource Packs menu
4. Ensure the GemPlugin is installed on your server

## Contents

### Textures
- `strength_gem.png` - Red gem texture for Strength Gem
- `fire_gem.png` - Orange gem texture for Fire Gem  
- `speed_gem.png` - Yellow gem texture for Speed Gem
- `astra_gem.png` - Purple gem texture for Astra/Puff Gem
- `ice_gem.png` - Blue gem texture for Ice Gem
- `invis_gem.png` - White gem texture for Invisibility Gem
- `gem_default.png` - Green default fallback gem texture

### Models
The pack includes a base model (`gem_base.json`) with custom_model_data overrides:
- 1001: Strength Gem
- 1002: Fire Gem
- 1003: Speed Gem
- 1004: Astra Gem
- 1005: Ice Gem
- 1006: Invisibility Gem

## Compatibility

- Minecraft Version: 1.21.x
- Pack Format: 64
- Server: Paper 1.21.8+

## Usage

Once the resource pack is enabled and the plugin is running, all gem items will automatically display their custom textures based on their custom_model_data values set by the plugin.
